# Agent 新手教程

这是一个面向完全新手的 Agent 教程站点。行文顺序是：

1. 从零理解 Agent。
2. 学会选择和购买 Agent 产品。
3. 以 Claude Code 为例入门 Coding CLI。
4. 理解类龙虾应用和 Hermes。
5. 用 Claude Code + Obsidian 做本地知识库。
6. 用 Hermes + 知识摄入 Skill 做自动知识摄取。

精确命令、订阅价格和平台差异会随时间变化，购买和安装前请回到官方链接确认。

## 本地预览
