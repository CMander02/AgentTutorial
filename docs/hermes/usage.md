# 类龙虾应用怎么用

类龙虾应用的核心使用方式不是“打开一次问一句”，而是把常见任务配置成入口和流程。

## 典型入口

- 网页对话框
- QQ / 微信 / 飞书 Bot
- 定时任务
- Webhook
- 手动上传内容

## 典型任务

- 每天抓取指定网页并总结。
- 群里发链接后自动整理到知识库。
- 每周生成回顾。
- 收到关键词后查询知识库并回复。
- 把手动输入的想法转成结构化笔记。

## 使用前准备

先在本地用 Coding CLI 把任务流程跑通，再把它交给 Hermes。否则你会同时调模型、调 Bot、调服务器和调知识库规则，排错会很困难。
