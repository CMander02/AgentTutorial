# 当前最热门的 Agent 产品形态

当前最热门的 Agent 产品形态，一是 Coding CLI ，二是类 OpenClaw Agent.

## 1. Coding CLI

2025年年中，随着 Cladue Sonnet 4 发布、配套的 Claude Code TUI 的发布与其第一次源码泄露，人们认识到，顶级 Coding LLM 能够依赖强大的 Agent 环境，在完全不依赖人类进行审批和检查的前提下，自主完成复杂庞大的代码项目。自此 Claude Code 深耕 TUI 非常久，现在已经是全球目前最强的 Agent 环境，任何开闭源模型，直接接入 Claude Code，都能获得极强的性能提升。

Claude Code 也反哺了 Claude 系列模型的训练：它帮助 Anthropics 获得了大量用户的代码数据和流程规划数据，巩固了 Skill 和 MCP 的地位，且作为稳定的框架参与进 Claude 系列模型的数据生产、模型训练和流程反馈、乃至整个 Anthropic 公司的研发流程中。

## 2. 类 OpenClaw Agent

2026开年，Github 开源榜一声炮响，给全球送来了赛博龙虾。

龙虾的贡献与不足前文已经叙述过，奈何 Claude Code, Codex 等模型厂自行研发的 Agent 环境，在那个时候仍然无法接入聊天软件。大量的用户转投 OpenClaw 的怀抱，把 Claude Code 和 Codex 包装成 LLM 接口，以在不同环境下使用 OpenClaw。

OpenClaw 的火热带动了大量类似开源项目的发展，目前类似项目中最热门的，是易用性和稳定性都更高的 Hermes。后续的部分教程也会基于 Hermes 展开。

## 3. 新手用哪一种？

纯新手推荐使用 Coding CLI，如 Claude Code CLI, OpenCode CLI, Codex CLI，或者自带开发完善的 GUI 的 Coding CLI（如 OpenCode GUI, Codex GUI 等）。

不建议在没有 Agent 基础、且不需要定时任务的前提下使用类 OpenClaw Agent。
