# 总结、分类、入库规则

自动摄取最容易失败的地方是分类过度。新手应先用少量稳定分类。

## 推荐状态

| 状态        | 含义             |
| ----------- | ---------------- |
| `raw`       | 刚摄入，未处理   |
| `inbox`     | 已摘要，但未归档 |
| `review`    | 需要人工确认     |
| `note`      | 已整理为长期笔记 |
| `discarded` | 不再保留         |

## 入库原则

- 不确定的内容不要进 `Notes/`。
- 来源必须保留。
- 摘要和观点分开。
- 自动分类只给建议，先不要强制移动。
- MOC 更新要链接真实存在的笔记。

## 去重规则

先按 URL 去重，再按标题和语义相似度人工确认。不要只因为标题相似就覆盖旧笔记。
