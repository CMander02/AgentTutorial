# 互联网内容摄取

互联网内容包括网页、RSS、论文、博客、视频字幕和社交媒体帖子。

## 输入格式

建议统一成：

```text
来源 URL
标题
抓取时间
正文或摘要
触发来源：手动 / 定时 / Bot
```

## 处理步骤

1. 抓取内容。
2. 提取正文。
3. 记录来源。
4. 判断是否重复。
5. 总结关键观点。
6. 写入 `Inbox/` 或 `Sources/`。

## 新手建议

不要一开始就抓所有来源。先选 3 到 5 个高质量来源，确认总结质量和分类规则稳定后再扩大。
