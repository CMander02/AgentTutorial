# 手动内容摄取

手动内容包括你自己写的想法、聊天记录、会议记录、截图 OCR、临时任务和读书摘录。

## 为什么需要手动入口

不是所有重要内容都来自网页。很多知识来自你的日常工作和即时想法。

## 推荐入口

- Hermes Bot 私聊
- 飞书群消息
- 手动粘贴到 Claude Code
- Obsidian Daily note
- 一个统一的 `Inbox/raw.md`

## 处理规则

手动内容不要求立刻整理成长期笔记。建议先做三件事：

1. 保存原文。
2. 提取一句话摘要。
3. 标记下一步：丢弃、待整理、可归档。

下一步：[总结、分类、入库规则](organize.md)。
