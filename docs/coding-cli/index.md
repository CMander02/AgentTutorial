# Coding CLI 入门：以 Claude Code 为例

Coding CLI 是新手最容易真正用起来的 Agent 形态。本章用 Claude Code 做主线，但很多方法也适用于 Codex CLI、Gemini CLI 等工具。

你会学到：

- 如何在本地安装 Claude Code。
- 如何和 Coding CLI 交互。
- 如何管理 `CLAUDE.md` / `AGENTS.md`。
- 如何用 `/plan`、`/goal` 和历史记录管理项目。
- Skill 和 MCP 分别是什么。
- 如何安装常用 Skill 和 MCP。
