# 常用 Skill 与使用思维

Skill 不是“多一个按钮”，而是把一套工作方法写给 Agent。新手要先理解使用思维，再追求安装更多 Skill。

## Brainstorm 型 Skill

适合早期探索：

- 想选题
- 想产品方案
- 想文档结构
- 想实验设计

使用思维：

```text
先发散，再收敛；先列选择标准，再选方案。
```

## Superpower 型 Skill

适合让 Agent 获得某类稳定能力，例如：

- 写 Obsidian 笔记
- 处理 spreadsheet
- 做 slides
- 画 JSON Canvas
- 操作浏览器

使用思维：

```text
把不稳定的临时 prompt 固化成可复用流程。
```

## 项目内 Skill

适合某个项目独有的流程：

- 周报格式
- 论文引用检查
- 数据发布流程
- 知识库整理规则

使用思维：

```text
只要团队成员会重复解释，就应该沉淀成项目级 Skill 或项目说明。
```
